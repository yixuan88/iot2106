IoT Mesh Gateway — Protocol Diagrams

Each diagram shows one protocol in isolation: how devices connect, what data moves, and in which direction.

📶

WiFi — Web UI & REST API

Devices connect to the RPi's WiFi Access Point (MeshGateway-<name> · 192.168.4.1) and use HTTP + WebSocket

🔵

Bluetooth LE — NUS GATT + Beacon

RPi advertises BLE beacon with mesh status. M5StickC / iPhone connect via Nordic UART Service (NUS) for bidirectional text

📨

MQTT — Pub/Sub Message Broker

Mosquitto broker on each RPi. WiFi clients publish/subscribe via WebSocket (:9001). mqtt_bridge connects internally on :1883.

📡

LoRa Mesh — Meshtastic Radio Network

4 × LILYGO T3 LoRa32 V1.6.1 form a self-healing mesh. Multi-hop routing, 228-byte packets, zone-tagged wire format.

🔌

USB Serial — RPi ↔ ESP32 (Meshtastic)

Each RPi connects to its LILYGO T3 LoRa32 via USB serial. mesh_interface wraps the Meshtastic Python library.
